skjermer
========

This program parses [the computer monitor category at Prisjakt.no](http://www.prisjakt.no/category.php?k=393).

Dependencies
------------

* [Python 3](https://www.python.org/)
* [Requests](http://docs.python-requests.org/en/latest/)
* [lxml](http://lxml.de/)

-------
